# MNIST トポロジカルデータ解析

MNISTデータセットにパーシステントホモロジーを適用した解析

## 概要
このノートブックでは、MNISTの手書き数字画像に対してパーシステントホモロジーを用いたトポロジー特徴抽出を行った例です。

## 使い方
1. main.ipynbをクリックしコード内容と結果を確認もしくは、ipynbファイルをダウンロードし、colabなどで実行。
2. Colabで開いたら、ランタイム → すべてのセルを実行

## 特徴量
- H1（1次元ホモロジー）: 画像中の「穴」の数と大きさ
- 距離変換を用いたフィルトレーション

## 結果
各数字（0-9）のトポロジー特徴の平均値を計算し、各データに対して正しくトポロジー的特徴量が抽出できた。
